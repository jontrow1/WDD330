Week 1 Notes

  1. Input tag = Single line of text.
  2. Textarea tag = Multi-line of text.
  3. Header tags start with 'h' and go from 1 - 6, no .5's.
  4. Section tags fill the screen all the way across
  5. Button tag = buttons, and with JavaScript, can actually do something
  6. The background color of any tage can be changed.
  7. It doesn't matter what order functions go in for JavaScript, they will all be read.
  8. var = variable (can also use 'let')
  9. When looking at a variable, the computer reads on the right hand of the equal sign first.
  10. When dealing with input or testarea, use 'value'; section, use 'innerHTML'
  11. localStorage is used to store for later use
  12. key-value pairs example: Student (key), Grades (value)
  13. When getItem in a key-value pair, you only need the key
